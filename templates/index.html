Демо лента новостей на Django

Необходимо реализовать ленту новостей, каждая новость принадлежит какой-нибудь категории (Политика, Бизнес, Спорт и прочее), у новости есть название, короткий текст (анонс), полный текст новости, дата публикации, изображение, а также теги. Должна быть возможность получить список новостей по категории и по выбранному тегу. В списке новостей показывается изображение, название новости, дата и анонс. При просмотре новости показываются все поля. Для создания новостей достаточно сделать простой админ с формой.

Требования к коду:
— Язык программирования любой на выбор PHP/Python/Perl/Ruby.
— СУБД – MySQL
— Базовая верстка и JavaScript при необходимости
— Применение библиотеки Bootstrap для построения интерфейсов (будет большим плюсом, но необязательно)
— Отформатирован
— Легко читаем

Срок выполнения – 7 дней.

Протестировано на Django 1.5.4, Python 2.7.5.
Запуск:
1) cd project_path
2) python manage.py syncdb
3) python manage.py runserver
В проекте использован Bootstrap v.3.1.0
Суперпользователь админки: test, пароль test.
